Componente selección

Type Select

Type listbox


Como utilizar

<app-seleccion [textSeleccion]="'Seleccione una ciudad'" [items]="itemsExampleSeleccion" [type]="'select' (selected)="itemSelected($event)" > <app-seleccion>

Propiedades:

  • [type]: Tipo de selecctor (select para un selector común y listbox para utilizar un componente primeng multiseleccion más potenciado)
  • [textSeleccion]: Para indicar el texto inicial que desee que aparezca dentro select.
  • [items]: Datos a desplegar para ser seleccionados

Funciones:

  • (selected): Callback que retorna el valor seleccionado (el code en caso de select común y un array en caso de listbox)

JSON de datos = itemsExampleSeleccion

{{ itemsExampleSeleccion | json }}
